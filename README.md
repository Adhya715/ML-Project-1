## üéì Key Project Features

* **End-to-End Integration**: Successfully bridged a data science model with a functional web interface via a Flask API.
* **Data Preprocessing**: implemented advanced data cleaning, outlier removal (PPSF), and dimensionality reduction.
* **One-Hot Encoding**: Handled over 200+ categorical location variables using dummy variables to maintain model accuracy.
* **Artifact Serialization**: Utilized Python's `pickle` and `json` libraries to save and load the trained model and column metadata efficiently.
* **CORS Enabled**: Configured custom response headers to allow cross-origin requests between the frontend and backend.
* **Modern UI**: Replaced standard tutorial styles with a sleek, user-friendly interface featuring segmented controls and background overlays.
